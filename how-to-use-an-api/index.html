Why do we need a sandbox?

A lot of the data work we do is specifically for our clients, and not all of it is open data. That makes it hard for us to share examples of what we do and how it works. This sandbox is where we'll showcase a few samples and proofs of concept to demonstrate the actual things we can build with data and code.

This visualisation is built from data in the Libraries Unlimited datastore (https://libraries.thedata.place/). One of the datasets is an aggregated and anonymised CSV file with the number of registered borrowers by single year of age, giving us a basic view of the age profile of people who use their local library for borrowing items. Many people use libraries for other reasons, especially where the facilities include a cafe, children's activities, study space and business support.

In this sample we're using the CKAN API to extract the data, a bit of Python to tidy it up and structure it, and the Plotly library to turn it into a simple bar chart.
